# 0bsrv.eu

Inspired by the design of [axiom.co](https://axiom.co/) and other observability tools, I created this website to visualize the state of Europe at a glance. The site uses the [eurostat API](https://ec.europa.eu/eurostat/web/main/home) and [Plot](https://observablehq.com/plot/) to display data.
